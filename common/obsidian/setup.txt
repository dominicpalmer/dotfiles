## Editor
  - Unset 'Fold heading'
  - Unset 'Fold indent'
  - Set 'Show line numbers'
  - Unset 'Show indentation guides'
  - Unset 'Spellcheck'
  - Unset 'Indent using tabs'

## Appearance
- 'Base color scheme' as 'Light'
- 'Text' font as 'Cascadia Code'
- Set 'Native menus'
- 'Window frame style' as 'Native frame'
- Unset 'Show tab title bar'
- Set zoom level and font size as needed

## Community Plugins
- Turn on community plugins via 'Community Plugins'
- Install 'Hider', enable, and hide everything except:
  - Scroll bars
  - File explorer buttons

## Custom CSS
- Copy obsidian.css into the .obsidian/snippets directory of the 'main' vault
- Enable CSS snippets via 'Appearance/CSS snippets'
